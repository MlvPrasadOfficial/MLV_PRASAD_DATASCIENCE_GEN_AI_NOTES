## üõ†Ô∏è Essential Functions & Methods

### Figure-Level Functions
```python
import seaborn as sns
import matplotlib.pyplot as plt

# Relational plots
sns.relplot(kind='scatter'), sns.relplot(kind='line')
sns.scatterplot(), sns.lineplot()

# Distribution plots
sns.displot(kind='hist'), sns.displot(kind='kde')
sns.histplot(), sns.kdeplot(), sns.ecdfplot()

# Categorical plots
sns.catplot(kind='strip'), sns.catplot(kind='box')
sns.boxplot(), sns.violinplot(), sns.barplot()
```

### Axes-Level Functions
```python
# Statistical plots
sns.regplot(), sns.residplot()
sns.heatmap(), sns.clustermap()

# Specialized plots
sns.jointplot(), sns.pairplot()
sns.facetgrid(), sns.pairgrid()
```

### Styling and Customization
```python
# Aesthetics
sns.set_style(), sns.set_context()
sns.set_palette(), sns.color_palette()

# Themes and colors
sns.set_theme(), sns.despine()
sns.palplot(), sns.blend_palette()
```

### Data Integration
```python
# Dataset loading
sns.load_dataset()

# Utility functions
sns.axes_style(), sns.plotting_context()
sns.blend_palette(), sns.light_palette()
```
