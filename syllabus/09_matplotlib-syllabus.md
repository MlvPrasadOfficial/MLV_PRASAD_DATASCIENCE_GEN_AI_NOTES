## üõ†Ô∏è Essential Functions & Methods

### Basic Plotting
```python
import matplotlib.pyplot as plt
import numpy as np

# Basic plots
plt.plot(), plt.scatter(), plt.bar()
plt.hist(), plt.boxplot(), plt.pie()

# Figure and axes
plt.figure(), plt.subplots()
plt.subplot(), plt.subplot2grid()

# Customization
plt.xlabel(), plt.ylabel(), plt.title()
plt.legend(), plt.grid(), plt.colorbar()
```

### Object-Oriented Interface
```python
# Figure and axes creation
fig, ax = plt.subplots()
fig.add_subplot(), fig.add_axes()

# Plotting methods
ax.plot(), ax.scatter(), ax.bar()
ax.hist(), ax.boxplot(), ax.imshow()

# Customization
ax.set_xlabel(), ax.set_ylabel()
ax.set_title(), ax.legend(), ax.grid()
```

### Advanced Features
```python
# Annotations and text
plt.annotate(), plt.text()
ax.annotate(), ax.text()

# Colors and styling
plt.style.use(), plt.cm
plt.colors, plt.colormaps

# Animation and interactivity
from matplotlib.animation import FuncAnimation
from matplotlib.widgets import Slider, Button
```

### Specialized Modules
```python
# 3D plotting
from mpl_toolkits.mplot3d import Axes3D
```
