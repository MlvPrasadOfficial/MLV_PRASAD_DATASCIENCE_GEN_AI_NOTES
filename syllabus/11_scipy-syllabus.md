## üõ†Ô∏è Essential Functions & Methods

### Linear Algebra
```python
from scipy import linalg

# Matrix operations
linalg.solve(), linalg.lstsq()
linalg.eig(), linalg.svd()
linalg.lu(), linalg.qr(), linalg.cholesky()

# Matrix functions
linalg.expm(), linalg.norm()
linalg.inv(), linalg.pinv()
```

### Statistics
```python
from scipy import stats

# Distributions
stats.norm, stats.t, stats.chi2
stats.fit(), stats.rvs(), stats.pdf()

# Tests
stats.ttest_ind(), stats.mannwhitneyu()
stats.kstest(), stats.shapiro()
```

### Optimization
```python
from scipy import optimize

# Minimization
optimize.minimize(), optimize.minimize_scalar()
optimize.differential_evolution()

# Root finding
optimize.root(), optimize.fsolve()
optimize.brentq(), optimize.newton()
```

### Signal Processing
```python
from scipy import signal

# Filters
signal.butter(), signal.filtfilt()
signal.savgol_filter()

# Analysis
signal.fft(), signal.spectrogram()
signal.find_peaks(), signal.correlate()
```

### Image Processing
```python
from scipy import ndimage

# Filtering
ndimage.gaussian_filter()
ndimage.median_filter()

# Morphology
ndimage.binary_erosion()
ndimage.label(), ndimage.center_of_mass()
```
